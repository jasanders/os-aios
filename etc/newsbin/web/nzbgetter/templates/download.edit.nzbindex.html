Edit download list entry




Title

Give the entry a usefull name. In case of NZBINDEX.nl matches and with Twitter support enabled it will 'tweet' this title.

Note: The title is not needed to identify releases in NZB feeds. It's just to show a fancy name in the download list.


Match pattern

The records found in the NZB RSS feeds should match this pattern. If it matches, it will download the attached NZB file.

- For NZBINDEX.nl records you CAN combine phrases.

For example:

NZBINDEX:
You can use patterns here as if you are searching on Google.


Download to NZB incoming folder or add to verification queue first?

If a NZB file is found you can choose if you want tot move it to the incoming folder for immediate download, or you can move it to the NZB verification queue. If you choose to move files to this queue you can verify the matches first and decide whether to download them or not.


NZB filename

This field is optional.

You can give NZB files matching this entry a static filename. (only available in auto download to nzb folder mode)

Use this option with caution! If multiple matches are found within the same run they will all get the same NZB filename (which means some files will be overwritten)


.nzb

Exclude keywords

Your download should match the patterns you specified in the 'Match pattern' section above. But you can exclude downloads which contain certain phrases or keywords. You can enter multiple keywords by using a comma to seperate them.


Minimum size

Matching items should at least be this amount of megabytes.


Maximum size

Matching items should not exceed this amount of megabytes. (optional)


Specific poster

Sometimes you want releases from a specific usenet poster. If you want all posts from a specific poster leave the match pattern field empty, it will then matches all the posts from this user. (optional)
