Settings

Your new settings are stored!

WGET full path (from root directory)

Primarily this script uses the PHP cURL library to download NZB files from various sources. But some systems and NAS solutions don't have PHP cURL library support enabled. If you don't have cURL enabled the script supports the shell command 'wget' to receive the NZB files.

Please click the 'check now' button to run a short benchmark on your system to determine which NZB download method to use.


NZB path (from root directory)

Please provide the full path to your usenet client it's incoming NZB directory. Please make sure this directory is writable and correct, otherwise your downloads wont start.

If you don't have an usenet client installed, consider using NZBGet which can be found at http://nzbget.sourceforge.net/


Filename for logfile

Provide a filename for the logfile. (default is nzbd.log)


Log maintenance

You can reset the logfile if it exceeds a specific number of lines.


TVNZB.com RSS feed

Full URL to the TVNZB.com RSS feed. Unless the URL changes please keep this setting unchanged.


TV Series quality identifiers

These strings identify the release as either a High Definition release or a normal XviD from the TVNZB.com RSS feed. The default settings should be ok, so you can keep this unchanged.

NZBINDEX.nl RSS feed

Full URL to the NZBINDEX.nl RSS feed. Unless the url changes you can keep this setting unchanged.


NZBINDEX.nl NZB file age

If a record from your download list is found in the NZBINDEX.nl feed it may not exceed this number of days. (this setting can be handy if the retention on your newsserver is low)


NZBINDEX.nl number of results

Maximum number of results a check on NZBINDEX.nl may return.


Tweet your downloads

When enabled, every download will be announced (tweeted) to the specified Twitter account.

Note: this feature only works with PHP cURL!

So if no tweets appear after a download list match your PHP cURL is probably disabled.


Twitter username

Your Twitter username. Tweets will be send to this account.


Twitter password

Enter the password for your Twitter account.


Twitter message

The Tweet message, you can use the following parameters:

%nzbgetter = The current NZBGetter title + version
%download = The title of the download
